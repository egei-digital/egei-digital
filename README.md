# EGEI Digital — Versión 8 · Módulo 1

## Incluye
- Nuevo panel administrativo profesional.
- Dashboard con métricas.
- Reloj digital en tiempo real para La Paz, Bolivia.
- Fecha actual automática.
- Menú lateral colapsable.
- Navegación adaptable a celulares.
- Indicador de sesión activa.
- Accesos rápidos.
- Gráfico de preinscripciones de los últimos siete días.
- Modo claro y oscuro.
- Corrección completa de contraste: en modo oscuro los textos son blancos o grises claros.
- Pantalla de acceso rediseñada.

## Acceso de demostración

- Contraseña: `egei2026`

## Alcance del Módulo 1
Este módulo construye la interfaz y navegación del panel. La administración funcional de cursos, fechas, banners y noticias corresponde al Módulo 2. La preinscripción integrada y la conexión con Excel Online corresponden al Módulo 3.

## Nota de seguridad
El acceso actual es demostrativo y funciona en el navegador. La autenticación segura se implementará al conectar Firebase o un servicio de backend.
